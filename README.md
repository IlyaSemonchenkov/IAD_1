# Лабораторная работа №1: Первичное исследование и оценка качества данных

## Описание проекта
Первичный анализ датасета "toyota.csv" в рамках дисциплины "Интеллектуальный анализ данных".

**Предметная область:** Датасет содержит информацию о подержанных автомобилях Toyota моделей GT86, Corolla, RAV4 и Yaris и т.д. с характеристиками: модель, год выпуска, цена, трансмиссия, пробег, тип топлива, налог, расход топлива (mpg), объем двигателя. Возможные задачи ИАД: предсказание цены автомобиля, сегментация по характеристикам, анализ факторов стоимости.

**Цель лабораторной:** Аудит качества данных, построение паспорта датасета, выявление проблем.

## Структура проекта
- `notebooks/01_data_understanding.ipynb` — основной Jupyter Notebook с анализом  
- `report/quality_report.md` — текстовый отчёт с выводами и планом улучшения  
- `data/` — датасет или скрипт загрузки
